This repository contains the Jac (Jaseci) backend for the AngaWatch MVP:
- Collects multi-source weather reads (WeatherRaw)
- Merges data into WeatherUnified
- Uses an LLM (byLLM) to generate human-friendly forecast (ForecastAI)
- Predicts simple RiskEvents (disaster detection)
- Dispatches notifications to users

> **Compatibility:** Written for Jaseci / jac 0.9.x (your environment). Confirm by running `jac --version`.
